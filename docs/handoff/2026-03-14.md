## 핵심 결정 & 맥락

- **nDRIMS 자동화 포기**: 2단계 인증(EZGuard) + CLX 프레임워크 보안 정책으로 headless 자동 로그인 불가. 로컬 headed 브라우저 수동 로그인 전용
- **공지 본문 항상 수집**: 이전에는 캐시 히트 시 본문 추출을 건너뛰어 재크롤링 시 본문 소실. 캐시 무관하게 항상 수집으로 변경
- **볼륨 마운트 통일**: 크롤러는 `/app/output`에 쓰고 data_loader도 `/app/output`에서 읽도록 통일
- **lifespan 패턴**: FastAPI deprecated `on_event` → `asynccontextmanager lifespan`으로 전환, auto-sync 태스크 graceful shutdown 지원
